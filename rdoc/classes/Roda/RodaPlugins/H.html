module Roda::RodaPlugins::H

  1. lib/roda/plugins/h.rb

The h plugin adds an h instance method that will HTML escape the input and return it.

The following example will return “&lt;foo&gt;” as the body.

plugin :h

route do |r|
  h('<foo>')
end
