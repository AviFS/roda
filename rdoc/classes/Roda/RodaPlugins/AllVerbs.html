module Roda::RodaPlugins::AllVerbs

  1. lib/roda/plugins/all_verbs.rb

The all_verbs plugin adds methods for http verbs other than get and post. The following verbs are added, assuming rack handles them: delete, head, options, link, patch, put, trace, unlink.

These methods operate just like Roda's default get and post methods, so using them without any arguments just checks for the request method, while using them with any arguments also checks that the arguments match the full path.

Example:

plugin :all_verbs

route do |r|
  r.delete do
    # Handle DELETE
  end
  r.put do
    # Handle PUT
  end
  r.patch do
    # Handle PATCH
  end
end

The verb methods are defined via metaprogramming, so there isn't documentation for the individual methods created.
