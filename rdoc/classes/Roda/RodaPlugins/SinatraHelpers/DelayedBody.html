each ()

If the body is a String, yield it, otherwise yield each string returned by calling each on the body.

[show source] 
# File lib/roda/plugins/sinatra_helpers.rb, line 237
def each
  v = value
  if v.is_a?(String)
    yield v
  else
    v.each{|s| yield s}
  end
end
